• 1995年にプロトタイプベースのオブジェクト指向言語として登場したが、2015年にリリースされた ECMAScript 6th Editionでは、classベースのオブジェクト指向プログラミングやアロー関数を新たにサポート
  • 狭義には Mozilla Foundation を仕様を策定しているスクリプト言語を指すが、ここではECMAScript 6以前のバージョンであるECMA-262 5.1 Edition および、ECMAScript 6（ECMA-262 6th Edition、ECMAScript 2015 とも呼ばれる）以降を指す

  • 1991年に登場した汎用プログラミング言語（コンパイル不要）
  • Python3系（最新版3.9.5）の他に2系（最終版2.7.18）があったが、2020年に2系は正式に廃止された
  • 標準を含めライブラリが充実
  • 記述には {} や ; が不要
  • インデントが重要な意味を持ち、同じ数の空白でインデントされた文がブロックとみなされる
  • Blenderで採用（Blender2.5以降はPython3系）

  • 2000年にアルファ版が登場、2002年に Ver.1.0 がリリース（現在 Ver.9.0）
  • C/C++ から構文・キーワード・演算子を譲り受けつつ、文法は Java に大きな影響を受けている
  • .NET Framework 、Mono等のライブラリを使うことで最大限の力を発揮する
  • Ecmaインターナショナル、ISOにより標準化され、JISで標準プログラミング言語として制定されている
  • ゲームエンジンのUnityでも採用
